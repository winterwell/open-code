The input for the simple (name list) gender detector class GenderDetect are in the form e.g.

namelist_en_gb_f.txt -> English, United Kingdom, female names

Everything else is source data of some sort.

UK names:

historicname_tcm77-254032.xls : Source data from Office of National Statistics
uk_*.csv : Source data saved as CSV sans header
compile_names.py : Takes *.csv and generates lists
